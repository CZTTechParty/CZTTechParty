简介
-------------------------------

CZTTechParty是由多名IT技术爱好者在长株潭地区组织的第一个以自由、开放、合作、共享为宗旨的线下技术交流平台。

CZTTechParty自2013年9月开始筹备成立以来得到众多网友的支持，我们真诚的欢迎更多的IT技术爱好者加入到分享活动中来。
